Faster RegEx extraction


  str = "GB is not Great Britain"

  # the usual way
  match_results = str.match /(GB)/
  match_results[1]
  => "GB"
Can be shortened to: 

    str[/(GB)/, 1]
    => "GB"

Unary operators

+, -, ! and ~ can be overriden


  class Counter
    def initialize
      @value = 0
    end

    def +@
      @value += 1
    end

    def -@
      @value -= 1
    end

    def ~
      @value
    end
  end
  

c = Counter.new
+c
+c
puts ~c # 2

Method Madness


  def m1(a, b=1)        # correct



  def m2(a=1, b)        # correct



  def m3(a, b=2, c)     # correct



  def m4(a=1, b, c=3)   # wrong (optional args must be sequential)



  def m5(a, b=2, *args) # correct



  def m6(a=1, b, *args) # wrong

Method Madness


  def m7(*args, c)      # correct
    

  def tail(*args, last)

  end

Modules

How do they work?


  class MyClass
  end



  MyClass.ancestors
  => [MyClass, Object]
      

  class MyClass
    include MyModule
  end



  MyClass.ancestors
  => [MyClass, MyModule, Object]
      

  class MyClass
    include MyModule
    include AnotherModule
  end



  MyClass.ancestors
  => [MyClass, MyModule,
         AnotherModule, Object]
